# 📝 To-Do List (React)

Uma aplicação simples e funcional de lista de tarefas, desenvolvida com React, com foco em organização, produtividade e boas práticas de desenvolvimento front-end.

---

## 🚀 Funcionalidades

- ✅ Adicionar novas tarefas
- ✏️ Editar tarefas existentes
- ❌ Remover tarefas
- 📌 Marcar tarefas como concluídas
- 💾 Persistência de dados (se tiver localStorage, pode manter)

---

## 🛠️ Tecnologias utilizadas

- React
- JavaScript (ES6+)
- HTML5
- CSS3
